## Summary

이 **개발노트**는 다음과 같이 구성하였습니다.

* Server - **Oracle Cloud**(free-tier)
    + WEB　Apache
    + WAS　Tomcat(embedded)
    * DB　　MySQL
* Front-end **Vue.js**
* Back-end **SpringBoot**
